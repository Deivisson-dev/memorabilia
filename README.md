# Memorabilia

**Memorabilia** é um projeto desenvolvido para a disciplina de Filosofia da Ciência utilizando **React**, **TypeScript**, **Vite**, **Tailwind CSS**, e componentes do **shadcn/ui**. O objetivo é criar um site que sirva de memorabilia para que possa ser armezenada as principais mémorias da disciplina de Filosofia da Ciência.
## Tecnologias Utilizadas

- **React**: Biblioteca JavaScript para construir interfaces de usuário.
- **TypeScript**: Superset de JavaScript que adiciona tipagem estática ao código.
- **Vite**: Ferramenta de build rápida e leve, com Hot Module Replacement (HMR).
- **Tailwind CSS**: Framework utilitário para estilização rápida e customizável.
- **shadcn/ui**: Componentes UI acessíveis e prontos para uso com uma boa integração em projetos React.



## Instalação e Configuração

1. Clone o repositório:

  ```bash
  git clone https://github.com/usuario/my-memorabilia.git
  ```

2. Navegue até o diretório do projeto:

  ```bash
  cd my-memorabilia
  ```

3. Instale as dependências:

  ```bash
  npm install
  ```

4. Rode o projeto em ambiente de desenvolvimento

  ```bash
  npm run dev
  ```
